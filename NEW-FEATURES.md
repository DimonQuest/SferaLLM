# Multi-Model-Chat-v2 - Новые улучшения

**Дата:** 2026-04-22

## Реализованные улучшения (сегодня)

### ✅ 1. История версий сообщений (Regenerate)
Добавлена возможность регенерации ответов от моделей с сохранением всех версий:
- Кнопка 🔄 для регенерации ответа ассистента
- Автоматическое сохранение предыдущих версий
- Навигация между версиями (◀ / ▶)
- Индикатор текущей версии (например, "2 / 3")
- Streaming поддержка при регенерации

**Файлы:**
- `src/types/index.ts` - добавлены `versions` и `currentVersionIndex` в Message
- `src/renderer/ChatWindow.tsx` - функции `regenerateMessage()` и `switchVersion()`
- `src/renderer/ChatWindow.css` - стили для кнопок версий

### ✅ 2. Экспорт в PDF/HTML
Расширены возможности экспорта чатов:
- **Экспорт в PDF** - через окно печати браузера с красивым форматированием
- **Экспорт в HTML** - сохранение в HTML файл с полным стилем
- Сохранение форматирования markdown
- Подсветка блоков кода
- Метаинформация о чатах (провайдер, модель, токены)

**Файлы:**
- `src/utils/pdfExport.ts` - утилиты экспорта в PDF/HTML
- `src/renderer/ExportImport.tsx` - добавлены кнопки экспорта

### ✅ 3. Система кастомных тем
Полноценная система управления темами оформления:
- **5 встроенных тем:** Dark, Light, Nord, Dracula, Monokai
- Создание пользовательских тем
- Редактор цветов с color picker
- Предпросмотр цветовой схемы
- Сохранение тем в localStorage
- Удаление пользовательских тем

**Файлы:**
- `src/types/theme.ts` - типы и менеджер тем
- `src/renderer/ThemeManager.tsx` - UI управления темами
- `src/renderer/ThemeManager.css` - стили
- `src/renderer/Settings.tsx` - интеграция в настройки

### ✅ 4. Система плагинов
Архитектура для расширения функциональности приложения:
- **4 типа плагинов:**
  - Provider - новые LLM провайдеры
  - UI - компоненты интерфейса
  - Middleware - обработчики запросов/ответов
  - Tool - дополнительные инструменты
- Установка плагинов из JSON файлов
- Включение/отключение плагинов
- Детальная информация о каждом плагине
- Фильтрация по типам
- Удаление плагинов

**Файлы:**
- `src/types/plugin.ts` - типы и PluginManager класс
- `src/renderer/PluginManager.tsx` - UI управления плагинами
- `src/renderer/PluginManager.css` - стили

## Как использовать новые функции

### История версий:
1. Отправьте сообщение модели
2. Нажмите 🔄 на ответе для регенерации
3. Используйте ◀ / ▶ для переключения между версиями

### Экспорт в PDF/HTML:
1. Откройте Экспорт/Импорт (Ctrl+E)
2. Выберите "Экспорт в PDF" или "Экспорт в HTML"
3. Для PDF откроется окно печати

### Кастомные темы:
1. Откройте Настройки (Ctrl+,)
2. Нажмите "🎨 Темы"
3. Выберите готовую тему или создайте свою
4. Настройте цвета в редакторе

### Плагины:
1. Создайте JSON файл с описанием плагина
2. Откройте менеджер плагинов (нужно добавить в меню)
3. Нажмите "Установить плагин"
4. Включите/отключите плагины переключателем

## Технические детали

### Версии сообщений:
- Хранятся в массиве `versions` внутри Message
- `currentVersionIndex` указывает на активную версию
- При регенерации старая версия сохраняется автоматически
- Поддержка streaming при регенерации

### Система тем:
- Темы хранятся в localStorage
- Применяются через CSS переменные
- Поддержка цветов, шрифтов и радиусов
- Разделение на встроенные и пользовательские

### Система плагинов:
- Singleton PluginManager для управления
- Плагины хранятся в localStorage
- Поддержка динамической загрузки (заготовка)
- Валидация при установке
