# Быстрый старт - Сборка для продакшна

## Команды для сборки

```bash
# Перейти в папку проекта
cd "D:\ii\omniroute\Multi-Model-Chat-v2"

# Установить зависимости (первый раз)
npm install

# Собрать для Windows
npm run dist:win
```

## Результат

Готовые файлы будут в папке `release/`:
- `Multi-Model LLM Chat Setup 2.0.0.exe` - установщик
- `Multi-Model LLM Chat 2.0.0.exe` - portable версия

## Проверка перед запуском

Убедитесь что установлен Node.js:
```bash
node --version  # должно быть >= 18.x
npm --version   # должно быть >= 9.x
```

Если нет - скачайте с https://nodejs.org/
