# SferaLLM - Multi-Model AI Chat Application

![Version](https://img.shields.io/badge/version-2.1.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20macOS%20%7C%20Linux-lightgrey)

Мощное десктопное приложение для работы с несколькими AI моделями одновременно. Поддержка 13+ провайдеров, прокси-серверов, мультиагентной коллаборации и многое другое.

## ✨ Основные возможности

### 🤖 Поддержка множества AI провайдеров
- **OpenAI** (GPT-4, GPT-3.5, GPT-4o)
- **Anthropic** (Claude 3/4 Opus, Sonnet, Haiku)
- **Google** (Gemini Pro, Gemini Vision)
- **Mistral AI** (Mistral Large, Medium, Small)
- **Cohere** (Command, Command Light)
- **Perplexity** (pplx-70b, pplx-7b)
- **Groq** (Llama 3, Mixtral)
- **xAI** (Grok)
- **DeepSeek** (DeepSeek Chat, Coder)
- **OpenRouter** (доступ к 100+ моделям)
- **OmniRoute** / **9router** (универсальный шлюз)
- **Custom** (любой OpenAI-совместимый API)
- **Локальные модели** (Ollama, LM Studio, LocalAI)

### 🌐 Система прокси (NEW!)
- **Без прокси** - прямое подключение
- **Автоматический прокси** - выбор из 21 страны
- **Ручной прокси** - собственный прокси с авторизацией

### 👥 Мультиагентная коллаборация
- **Normal Mode** - параллельная работа всех AI
- **Collaborative Mode** - 3 раунда обсуждения между AI
- **Orchestrator Mode** - координация задач главным AI

### 💬 Продвинутый чат
- Параллельная работа с несколькими моделями
- История сообщений с закладками
- Версионирование ответов
- Экспорт в Markdown, JSON, PDF
- Поиск по истории
- Сравнение ответов разных моделей

### 🎨 Интерфейс
- Темная и светлая темы
- Настраиваемые раскладки (горизонтальная, вертикальная, сетка)
- Drag & drop для изменения порядка чатов
- Голосовой ввод
- Прикрепление файлов (изображения, документы)

### 🔧 Дополнительно
- Шаблоны промптов
- Аналитика использования токенов
- Автосохранение
- Блокировка экрана с паролем
- Безопасное хранение API ключей
- Автообновление

## 📥 Установка

### Windows
1. Скачайте `SferaLLM-Setup-2.1.0.exe` из [Releases](https://github.com/DimonQuest/SferaLLM/releases)
2. Запустите установщик
3. Следуйте инструкциям мастера установки

**Portable версия:**
- Скачайте `SferaLLM-2.1.0-portable.exe`
- Запустите без установки

### macOS
1. Скачайте `SferaLLM-2.1.0.dmg`
2. Откройте DMG файл
3. Перетащите SferaLLM в папку Applications

### Linux
**AppImage:**
```bash
chmod +x SferaLLM-2.1.0.AppImage
./SferaLLM-2.1.0.AppImage
```

**Debian/Ubuntu:**
```bash
sudo dpkg -i sferallm_2.1.0_amd64.deb
```

## 🚀 Быстрый старт

### 1. Создайте первый чат
1. Нажмите **"+ Новый чат"**
2. Выберите провайдера (например, OpenAI)
3. Введите API ключ
4. Выберите модель (например, gpt-4)
5. Нажмите **"Создать"**

### 2. Настройте прокси (опционально)
В окне создания чата:
1. Прокрутите до **"🌐 Настройки прокси"**
2. Выберите режим:
   - **Автоматический** - выберите страну
   - **Ручной** - введите хост, порт, логин, пароль

### 3. Отправьте сообщение
Введите текст и нажмите Enter или кнопку отправки.

## 📖 Подробное руководство

### Получение API ключей

**OpenAI:**
1. Зарегистрируйтесь на https://platform.openai.com
2. Перейдите в API Keys
3. Создайте новый ключ

**Anthropic:**
1. Зарегистрируйтесь на https://console.anthropic.com
2. Перейдите в API Keys
3. Создайте новый ключ

**Google (Gemini):**
1. Перейдите на https://makersuite.google.com/app/apikey
2. Создайте API ключ

**Другие провайдеры:**
- Mistral: https://console.mistral.ai
- Cohere: https://dashboard.cohere.com
- Groq: https://console.groq.com
- OpenRouter: https://openrouter.ai/keys

### Использование локальных моделей

**Ollama:**
1. Установите Ollama: https://ollama.ai
2. Запустите модель: `ollama run llama3`
3. В SferaLLM создайте чат:
   - Провайдер: **Custom**
   - Base URL: `http://localhost:11434`
   - Модель: `llama3`

**LM Studio:**
1. Установите LM Studio: https://lmstudio.ai
2. Загрузите модель
3. Запустите Local Server
4. В SferaLLM:
   - Провайдер: **Custom**
   - Base URL: `http://localhost:1234`
   - Модель: имя загруженной модели

### Мультиагентная коллаборация

1. Создайте минимум 2 чата с разными моделями
2. Нажмите **"📢 Общий чат"**
3. Выберите режим:
   - **Normal** - все AI отвечают параллельно
   - **Collaborative** - AI обсуждают между собой
   - **Orchestrator** - один AI координирует остальных
4. Отправьте вопрос

**Примеры задач для коллаборации:**
- "Как лучше спроектировать систему аутентификации?"
- "Создай план разработки мобильного приложения"
- "Предложи архитектуру для микросервисов"

### Работа с прокси

**Автоматический режим:**
- Система автоматически найдет рабочий публичный прокси
- Выберите страну для геолокации
- Прокси кэшируется на 5 минут

**Ручной режим:**
- Используйте собственный прокси
- Поддержка авторизации (логин/пароль)
- Рекомендуется для продакшена

**Доступные страны:**
США, Германия, Франция, Великобритания, Нидерланды, Канада, Япония, Сингапур, Австралия, Бразилия, Индия, Россия, Украина, Польша, Испания, Италия, Швеция, Норвегия, Финляндия, Дания

## ⚙️ Настройки

### Общие настройки
- **Тема** - светлая/темная
- **Размер шрифта** - маленький/средний/большой
- **Длина ответа** - короткий/стандартный/детальный
- **Автосохранение** - интервал сохранения

### Безопасность
- **Блокировка экрана** - пароль и автоблокировка
- **Безопасное хранилище** - шифрование API ключей

### Голосовой ввод
- **Локальный** - распознавание в браузере
- **Whisper API** - через OpenAI API

## 🔧 Горячие клавиши

- `Ctrl+N` - Новый чат
- `Ctrl+E` - Экспорт/Импорт
- `Ctrl+T` - Шаблоны промптов
- `Ctrl+,` - Настройки
- `Ctrl+F` - Поиск по истории
- `Ctrl+D` - Сравнение ответов
- `Ctrl+A` - Аналитика
- `Ctrl+B` - Показать/скрыть панель чатов
- `F12` - Инструменты разработчика
