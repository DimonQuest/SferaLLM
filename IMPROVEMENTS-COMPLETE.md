## Горячие клавиши

- **Ctrl+1-9** - переключение между чатами
- **Ctrl+Enter** - отправка сообщения
- **Ctrl+B** - показать/скрыть панель чатов
- **Ctrl+T** - шаблоны промптов
- **Ctrl+E** - экспорт/импорт
- **Ctrl+A** - аналитика
- **Ctrl+F** - поиск по истории
- **Ctrl+D** - сравнение ответов
- **Ctrl+N** - новый чат
- **Ctrl+,** - настройки

## Технические детали

### Новые зависимости
Все функции реализованы без дополнительных npm пакетов, используя:
- Web Speech API (встроенный)
- Crypto API (встроенный)
- Intersection Observer API (встроенный)
- Electron safeStorage (встроенный)

### Структура новых файлов

```
src/
├── main/
│   └── secureStorage.ts          # Шифрование API ключей
├── renderer/
│   ├── SearchHistory.tsx         # Поиск по истории
│   ├── SearchHistory.css
│   ├── CompareResponses.tsx      # Сравнение ответов
│   ├── CompareResponses.css
│   ├── LockScreen.tsx            # Экран блокировки
│   ├── LockScreen.css
│   ├── MessageContent.tsx        # Рендеринг markdown/кода
│   ├── MessageContent.css
│   ├── VoiceInput.tsx            # Голосовой ввод
│   ├── VoiceInput.css
│   └── VirtualizedMessages.tsx   # Виртуализация списков
└── utils/
    ├── autoSave.ts               # Автосохранение
    ├── messagePagination.ts      # Пагинация сообщений
    └── performanceHooks.ts       # Хуки оптимизации
```
