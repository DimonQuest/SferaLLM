# Инструкция по запуску Multi-Model LLM Chat

## Если возникают проблемы с установкой Electron

### Вариант 1: Ручная установка

1. Откройте PowerShell или CMD от имени администратора
2. Перейдите в папку проекта:
```
cd "D:\ii\omniroute\Multi-Model-Chat-Clean"
```

3. Удалите node_modules (если заблокирован - перезагрузите компьютер):
```
rmdir /s /q node_modules
```

4. Установите зависимости:
```
npm install
```

5. Соберите проект:
```
npm run build
```

6. Запустите:
```
npm start
```

### Вариант 2: Использование Tauri вместо Electron

Если Electron продолжает вызывать проблемы, можно переключиться на Tauri (более легковесный).

### Вариант 3: Веб-версия

Можно запустить как веб-приложение:
```
npm run build:renderer
npx webpack serve
```

Откройте http://localhost:3000 в браузере.
