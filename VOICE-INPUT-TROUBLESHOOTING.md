# Решение проблемы с голосовым вводом

**Дата:** 2026-04-22

## Проблема

Ошибка `Speech recognition error: network` возникает потому что:
1. Web Speech API требует интернет-соединения к серверам Google
2. Electron может блокировать эти запросы
3. Firewall или антивирус могут блокировать доступ

## Решения

### Вариант 1: Проверить интернет и разрешения

1. **Проверьте интернет-соединение**
   - Откройте браузер и проверьте доступ к google.com

2. **Проверьте Firewall**
   - Windows Defender → Разрешить приложение через брандмауэр
   - Найдите Electron и разрешите доступ к сети

3. **Проверьте разрешения микрофона**
   - Windows → Настройки → Конфиденциальность → Микрофон
   - Включите доступ для приложений

4. **Перезапустите приложение**
   ```cmd
   cd D:\ii\omniroute\Multi-Model-Chat-v2
   npm run build
   npm start
   ```

### Вариант 2: Использовать Chrome вместо Electron

Web Speech API лучше работает в Chrome:

1. Откройте Chrome
2. Перейдите на `chrome://flags/#unsafely-treat-insecure-origin-as-secure`
3. Добавьте `http://localhost:8080` (если используете dev-сервер)
4. Перезапустите Chrome

### Вариант 3: Локальная запись (без распознавания)

Создан компонент `VoiceInputLocal.tsx` который:
- ✅ Записывает аудио локально
- ✅ Показывает уровень звука
- ✅ Не требует интернета
- ❌ Не распознает речь автоматически

Для использования замените в ChatWindow.tsx:
```typescript
import VoiceInputLocal from './VoiceInputLocal';

// Вместо VoiceInput используйте:
<VoiceInputLocal onTranscript={handleVoiceTranscript} />
```

### Вариант 4: Установить локальный сервер распознавания

Для полноценного локального распознавания нужен отдельный сервер:

**Whisper (OpenAI)** - лучший вариант:
```bash
pip install openai-whisper
```

Создайте локальный API сервер для распознавания и подключите к приложению.

## Рекомендация

**Самое простое решение:**
1. Проверьте интернет
2. Отключите Firewall временно для теста
3. Перезапустите приложение

Если не помогает - используйте Chrome для тестирования голосового ввода.

## Альтернатива

Если голосовой ввод критически важен, рассмотрите:
- **Whisper API** (платный, но работает отлично)
- **Google Cloud Speech-to-Text** (платный)
- **Локальный Whisper** (бесплатный, но требует настройки)
