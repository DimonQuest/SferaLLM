# 🚀 Автоматическая сборка Multi-Model Chat

## Для пользователя (самый простой способ)

### Вариант 1: Двойной клик (рекомендуется)

1. Найдите файл `build-auto.bat` в папке проекта
2. **Кликните правой кнопкой** → **Запустить от имени администратора**
3. Дождитесь завершения (5-10 минут)
4. Готово! Файлы будут в папке `release/`

### Вариант 2: PowerShell (альтернатива)

1. Найдите файл `build-auto.ps1`
2. **Кликните правой кнопкой** → **Запустить с PowerShell**
3. Дождитесь завершения
4. Готово!

## Что делает скрипт автоматически

✅ Проверяет наличие Node.js
✅ Скачивает и устанавливает Node.js (если нужно)
✅ Устанавливает все зависимости
✅ Компилирует TypeScript
✅ Собирает Electron приложение
✅ Создает установщик и portable версию
✅ Открывает папку с результатами

## Результат

После завершения в папке `release/` будут:

```
release/
├── Multi-Model LLM Chat Setup 2.0.0.exe    # Установщик (~100 MB)
└── Multi-Model LLM Chat 2.0.0.exe          # Portable (~180 MB)
```

## Время выполнения

- **Первый запуск** (с установкой Node.js): ~10-15 минут
- **Повторная сборка**: ~3-5 минут

## Если что-то пошло не так

### Ошибка: "Не удается выполнить скрипт"

**PowerShell блокирует скрипты:**
```powershell
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
```

Затем запустите `build-auto.ps1` снова.

### Ошибка: "Требуются права администратора"

Кликните правой кнопкой → **Запустить от имени администратора**

### Ошибка при скачивании Node.js

Скачайте вручную: https://nodejs.org/dist/v20.11.1/node-v20.11.1-x64.msi
Установите, затем запустите скрипт снова.

### Ошибка "npm install failed"

Попробуйте:
```bash
npm cache clean --force
npm install
```

## Ручная сборка (если автоматика не работает)

```bash
# 1. Установите Node.js с https://nodejs.org/

# 2. Откройте PowerShell в папке проекта
cd "D:\ii\omniroute\Multi-Model-Chat-v2"

# 3. Установите зависимости
npm install

# 4. Соберите приложение
npm run dist:win
```

## Для разработчика

### Быстрая пересборка (без установки зависимостей)

```bash
npm run dist:win
```

### Сборка без упаковки (для тестирования)

```bash
npm run pack
```

Результат в `release/win-unpacked/`

### Очистка и полная пересборка

```bash
# PowerShell
Remove-Item -Recurse -Force node_modules,dist,release -ErrorAction SilentlyContinue
npm install
npm run dist:win
```

## Распространение

### Для конечных пользователей:

**Установщик** - рекомендуется для большинства:
- Автоматическая установка
- Ярлыки в меню Пуск
- Автоматическое удаление

**Portable** - для продвинутых пользователей:
- Не требует установки
- Можно запускать с флешки
- Все настройки в одной папке
