# Функция прокси для чатов

## Описание

Добавлена полная поддержка прокси-серверов для всех чатов с тремя режимами работы:

### Режимы прокси

1. **Без прокси** (по умолчанию)
   - Прямое подключение к AI провайдерам
   - Без промежуточных серверов

2. **Автоматический прокси**
   - Автоматический выбор рабочего публичного прокси
   - Выбор страны из списка (США, Германия, Франция, и др.)
   - Автоматическая проверка работоспособности
   - Кэширование на 5 минут

3. **Ручной прокси**
   - Ввод собственного прокси-сервера
   - Поддержка авторизации (логин/пароль)
   - Полный контроль над настройками

## Использование

### Создание/редактирование чата

1. Откройте модальное окно создания/редактирования чата
2. Прокрутите до секции "🌐 Настройки прокси"
3. Выберите режим прокси:

#### Автоматический режим
- Выберите "Автоматический прокси (публичный)"
- Выберите страну из выпадающего списка
- Система автоматически найдет рабочий прокси

#### Ручной режим
- Выберите "Ручной ввод прокси"
- Введите:
  - Хост прокси (например, 123.45.67.89)
  - Порт (например, 8080)
  - Логин (опционально)
  - Пароль (опционально)

## Технические детали

### Архитектура

```
ChatWindow.tsx
    ↓ (использует fetchWithProxy)
proxyFetch.ts
    ↓ (получает прокси)
proxyManager.ts
    ↓ (настраивает Electron)
main.ts (IPC handlers)
    ↓ (session.setProxy)
Electron Session
```

### Файлы

1. **src/types/index.ts**
   - Добавлены поля прокси в `ChatConfig`
   - Добавлен `proxy` API в `ElectronAPI`

2. **src/utils/proxyManager.ts** (новый)
   - Получение публичных прокси
   - Тестирование прокси
   - Управление списком стран

3. **src/utils/proxyFetch.ts** (новый)
   - Обертка над fetch с поддержкой прокси
   - Кэширование прокси
   - Интеграция с Electron IPC

4. **src/renderer/ChatList.tsx**
   - UI для настройки прокси
   - Выбор режима и страны
   - Поля для ручного ввода

5. **src/renderer/ChatWindow.tsx**
   - Использование `fetchWithProxy` вместо `fetch`
   - Передача конфигурации чата

6. **src/main/main.ts**
   - IPC обработчики `set-proxy` и `clear-proxy`
   - Настройка Electron session
   - Обработка авторизации прокси

7. **src/main/preload.ts**
   - Экспорт proxy API в renderer process

### Как это работает

1. **Автоматический режим:**
   - При отправке сообщения вызывается `fetchWithProxy`
   - Система проверяет кэш прокси для выбранной страны
   - Если нет в кэше - запрашивает список публичных прокси
   - Тестирует каждый прокси (до 10 штук)
   - Первый рабочий прокси кэшируется на 5 минут
   - Настраивает Electron session через IPC
   - Все последующие запросы идут через прокси

2. **Ручной режим:**
   - Использует указанные пользователем настройки
   - Настраивает Electron session с авторизацией
   - Все запросы идут через указанный прокси

3. **Без прокси:**
   - Очищает настройки прокси в Electron session
   - Прямое подключение к API

### Источники публичных прокси

- ProxyScrape API
- Proxy-List Download API

### Безопасность

- API ключи не передаются через прокси в открытом виде
- Поддержка HTTPS для шифрования трафика
- Авторизация прокси через Electron session
- Пароли хранятся в конфигурации чата (localStorage)

## Ограничения

1. **Публичные прокси:**
   - Могут быть медленными
   - Могут быть нестабильными
   - Не гарантируется приватность
   - Рекомендуется для тестирования

2. **Рекомендации:**
   - Для продакшена используйте платные прокси (ручной режим)
   - Проверяйте работоспособность прокси
   - Не используйте публичные прокси для конфиденциальных данных

## Отладка

Логи прокси выводятся в консоль:
- `Using cached proxy: X.X.X.X` - используется кэшированный прокси
- `Fetching working proxy for country: XXX` - поиск прокси
- `Found working proxy: X.X.X.X` - найден рабочий прокси
- `✓ Proxy configured: X.X.X.X:XXXX` - прокси настроен
- `✓ Proxy cleared` - прокси отключен

Откройте DevTools (F12) для просмотра логов.

## Примеры использования

### Обход блокировок
Если OpenAI API заблокирован в вашей стране:
1. Создайте чат с OpenAI
2. Включите автоматический прокси
3. Выберите страну (например, США)
4. Отправьте сообщение

### Использование платного прокси
Если у вас есть платный прокси:
1. Создайте чат
2. Включите ручной режим прокси
3. Введите данные прокси (хост, порт, логин, пароль)
4. Сохраните чат

### Тестирование из разных стран
Для тестирования геолокации:
1. Создайте несколько чатов
2. Настройте разные страны для каждого
3. Сравните ответы
