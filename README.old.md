# Multi-Model LLM Chat v2.0

Десктопное приложение для параллельной работы с несколькими LLM моделями одновременно.

## Быстрый запуск

Дважды кликните на файл `start.bat` в корне проекта.

Или через командную строку:
```bash
cd "D:\ii\omniroute\Multi-Model-Chat-v2"
npm start
```

## Что нового в v2.0

### Основные функции
✅ **Streaming ответов** - текст появляется по мере генерации с кнопкой остановки  
✅ **Поиск по истории** - поиск по всем сообщениям с фильтрами и закладками  
✅ **Сравнение ответов** - side-by-side сравнение двух моделей с оценкой качества  
✅ **Syntax highlighting** - подсветка кода с кнопкой копирования  
✅ **Голосовой ввод** - Speech-to-Text для ввода сообщений голосом  

### Безопасность
✅ **Мастер-пароль** - защита доступа к приложению  
✅ **Автоблокировка** - блокировка после неактивности (1-60 мин)  
✅ **Шифрование** - безопасное хранение API ключей  

### Автоматизация
✅ **Автосохранение** - резервное копирование каждые 30 секунд  
✅ **Восстановление** - восстановление данных после сбоя  
✅ **Черновики** - автосохранение незавершенных сообщений  

### UI/UX
✅ **Drag & drop** - изменение порядка чатов перетаскиванием  
✅ **Размер шрифта** - настройка размера текста (маленький/средний/большой)  
✅ **Закладки** - отметка важных сообщений звездочкой  

### Производительность
✅ **Виртуализация** - быстрый рендер больших списков сообщений  
✅ **Ленивая загрузка** - загрузка истории по требованию  
✅ **Оптимизация** - мемоизация и батчинг обновлений  

## Горячие клавиши

- **Ctrl+1-9** - переключение между чатами
- **Ctrl+Enter** - отправка сообщения

- **Ctrl+T** - шаблоны промптов
- **Ctrl+E** - экспорт/импорт
- **Ctrl+A** - аналитика
- **Ctrl+F** - поиск по истории
- **Ctrl+D** - сравнение ответов
- **Ctrl+N** - новый чат
- **Ctrl+,** - настройки

## Особенности

### Streaming ответов
Ответы от моделей появляются в реальном времени по мере генерации. Можно остановить генерацию в любой момент кнопкой "⏹️ Остановить".

### Поиск по истории
Мощный поиск по всем сообщениям с фильтрацией по:
- Провайдеру и модели
- Дате (от/до)
- Типу сообщения (пользователь/ассистент)
- Закладкам

### Сравнение ответов
Сравнивайте ответы двух разных моделей side-by-side:
- Подсветка различий
- Оценка качества (1-5 звезд)
- Статистика изменений
- Навигация по истории

### Безопасность
- Мастер-пароль с SHA-256 хешированием
- Автоблокировка после неактивности
- Шифрование API ключей через electron safeStorage

### Голосовой ввод
Используйте голос для ввода сообщений:
- Поддержка русского языка
- Визуальный индикатор записи
- Отображение транскрипции в реальном времени

## Производительность

Приложение оптимизировано для работы с большими объемами данных:
- Виртуализация списков (рендер только видимых элементов)
- Ленивая загрузка истории (батчи по 50 сообщений)
- Мемоизация вычислений
- Debounce и throttle для частых обновлений
